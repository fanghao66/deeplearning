# conv2d
```python
torch.nn.Conv2d(
    in_channels, # 输入通道
    out_channels, # 输出通道
    kernel_size, # kernel 大小
    stride=1, # 步长
    padding=0, # padding
    dilation=1, # 膨胀系数
    groups=1, # 分组
    bias=True, # 偏置
    padding_mode='zeros', # padding mode 
    device=None, # 设备
    dtype=None # 数据类型
    )
```
